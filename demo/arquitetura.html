Cross-cutting concerns

Camadas que atravessam toda a arquitetura

Autenticação

JWT (RS256) com kid rotativo trimestral. Refresh token rotativo + revogação por dispositivo. mTLS com bancos.

Filtros AJAX

Listagens com debounce 250ms, paginação cursor-based, ordenação multi-coluna. URL state preservada.

Load balancing

Anycast da Cloudflare + health checks nos web services dos bancos. Failover automático em < 2s.

Observabilidade

Sentry para erros + traces, Cloudflare Analytics agregado, logs estruturados (JSON) com trace_id ponta-a-ponta.
